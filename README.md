# Sistema de gestión de biblioteca

En este repositorio se tiene el desarrollo del sistema de gestión de bibliotecas para la renta de libros.

## Tech Stack

- Angular
- Node.js
- Express
- MySQL

## Features

- Login
- Inicio
- Administración
- Autores
- Secciones
- Libros
- Prestamos
